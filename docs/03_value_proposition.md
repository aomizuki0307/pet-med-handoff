# 03. 価値提案と差別化

## 価値提案（1文）

高齢・慢性疾患の犬猫について、家族全員が「誰が・いつ・何を投与したか」を1タップで共有し、
飲ませ忘れと二重投薬を防ぐAndroidアプリ。

## 競合との差別化（1文）

ペットノートPlus等の「広く記録する」健康手帳と違い、**投薬の実行確認だけ**に絞り、
書き換え不可の監査履歴と二重投薬警告という「共有カレンダーやLINEでは作れない安全装置」を提供する。

## 対象（docs/01再掲）

毎日投薬がある犬猫 × 2人以上で世話を分担する日本の世帯。健康なペットの日記ユーザーは対象外。

## ASO候補語（DDレポートF-5）

主軸: `ペット 投薬 管理` / `猫 薬 記録` / `犬 薬 飲み忘れ` / `二重投薬 防止`
補助: `ペット 健康記録 家族共有` / `猫 薬 飲ませたか`
避ける: `ペット 健康管理`（単独では強競合と正面衝突）

## 説明文の言い切り（ストア/LP共通トーン）

- 「診断はしません。獣医師の指示どおり飲ませられたかを、家族で確認するための道具です」
- 「記録は書き換えられません。だから通院時にそのまま見せられます」
